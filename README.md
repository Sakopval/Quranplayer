# Definition of the app:
This app is for people who likes listening to quran but regular sound players is not enough for them.
## Features:
- It makes saving listener's progress easier as regular sound players caches it so deleting progress can happen but here it saves it in a file.
- Contains a session button for starting a timer.
- Contains a mushaf as a third tab to listen and read at the same time.
